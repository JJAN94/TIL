### IDE (Integrated Development Environment)

- 프로그래밍을 위한 통합 개발 환경
- 프로그래밍 시 필요한 각종 tool들이 유기적으로 통합되어 제공 된다.
  - 컴파일 / 빌드 / 디버깅 / 프로파일링 / 배포
- IDE를 사용하지 않고 텍스트 에디터에 Plugin이나 Extension 형태의 추가 툴을 설치하여 비슷한 환경 제공 가능
